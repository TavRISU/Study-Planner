# Study Planner — Flask Web App

Study Planner adalah aplikasi web berbasis Python (Flask) yang dirancang untuk membantu pelajar dari SMP, SMA, hingga mahasiswa dalam merencanakan studi, mengelola tugas, memantau waktu belajar, dan mendapatkan rekomendasi arah pendidikan serta karier.

## Fitur Utama
- Autentikasi pengguna (Register, Login)
- Manajemen tugas (Task Manager)
- Pelacakan waktu belajar (Study Sessions)
- Timer belajar (Pomodoro)
- Dashboard statistik otomatis
- Rekomendasi belajar & karier
- Notifikasi browser
- Penyimpanan dengan SQLite

## Cara Menjalankan (Local/Colab)
1. Install dependencies:
pip install -r requirements.txt

2. Jalankan server:
python app.py
Salin kode

3. Akses melalui browser di:
http://localhost:5000

## Teknologi
- Python 3.x
- Flask
- SQLite
- APScheduler
- Waitress (Production server)
